# Nouvelle Vague Intelligence

A new wave in intelligence.

The official website for Nouvelle Vague Intelligence — a static site (HTML, CSS, vanilla JS).

## Structure

- `index.html` — home
- `members.html` — the team
- `license.html` — license & usage
- `nv.css` — styles
- `nv.js`, `wave-bg.js` — scroll reveal + animated wave background
- `assets/` — logo and images

## Running locally

It's a static site — open `index.html` in a browser, or serve the folder:

```
python -m http.server 8000
```

Then visit http://localhost:8000.
